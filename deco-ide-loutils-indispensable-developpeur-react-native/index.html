Deco IDE - L'outil indispensable du développeur React Native

El Mehdi

Deco IDE est un IDE (Environnement de développement intégré) spécialement conçu pour la création des applications React Native

Voila la vidéo de présentation de Deco IDE :

React Native est un framework frontend qui a été open-sourcée par Facebook mi-2013 et est le fruit d’un travail commun entre les équipes de Facebook et d’Instagram permettant de générer des composants web au travers d’une API qui se veut simple.

Voila la vidéo de présentation de React Native :

Deco IDE vous permet de créer rapidement l'accès aux library React directement deuis l'IDE. il est aussi possible d'obtenir un aperçu via l'emulateur iOS ou Android
