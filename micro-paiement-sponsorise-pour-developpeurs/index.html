Micro paiement Bitcoin Sponsorisé pour Développeurs & Webmasters

El Mehdi

Ce ci est une API très simple en JSON RPC 2 composés seulement de deux appels. Elle vous permet de proposer des solutions de paiement sponsorisées pour monétiser l'accès à vos services et contenus sur Internet, application et téléphones mobiles, pour des Jeux, des articles, des vidéos, donation, contenus payants, recharge du compte du client, etc.

Totalement gratuit pour votre client.

Votre client visite ibtc.cash, et récupère un code d’une valeur en satoshi. Il ne vous reste plus qu’a associer ce code a votre adresse bitcoin par un simple appel JSON RPC.

  1. Sans inscription ou approbation du compte, vous pouvez l’utiliser immédiatement. Elle peut être intégrée dans les minutes à venir dans votre système.
  2. Les retraits, son automatique une fois par jour. Ils se passent en bitcoin, et la somme minimale d’envois est de 5 centimes (10 000 satoshis)
  3. Votre client n’a pas besoin de posséder ou même de connaître le bitcoin pour effectuer un paiement sur votre plateforme. Tout ce qu’il lui faut, est le code récupéré sur ibtc.cash.

Utilisation

Le premier appel prend deux paramètres, votre adresse bitcoin et le code du client qu’il a récupéré sur ibtc.cash. Cet appel va associer son code à votre adresse bitcoin.
Le second appel prend juste votre adresse bitcoin en paramètre, est vous retourne toutes les informations à propos d’elle, comme les sommes due, les sommes déjà payé, les codes utiliser, etc.

Dans le repository Github ce trouve des exemples d’utilisation en php et jQuery. Si vous utiliser un autre langage de programmation, vous aurait aucun mal a trouver un JSON RPC 2 CLIENT ici.

Pour voir le fonctionnement de l’API en cas réel, vous pouvez visiter la page « bitcoin gratuit » dans le portefeuille mircoin.com.
