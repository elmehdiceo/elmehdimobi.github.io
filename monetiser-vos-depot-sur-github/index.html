Monétiser vos Dépôts sur GitHub avec Git.market

El Mehdi

Si vous vous trouvez sur cet article, je peux supposer que vous chercher a Monétiser vos dépots GitHub ou Bitbucket.

Pour ceux qui ne connaissent pas, GitHub est un service de gestion de version utilisé par les développeurs pour collaborer plus facilement sur les projets.

Dans ce petit article je vais vous parler plus précisément du service git.market .

git.market c'est un marketplace pour vendre du code avec Git, permettant de vendre vos dépots privé.

git.market est un service innovant dont le concept est assez simple: après la mise en ligne d'un : "code/package/snippet/theme/template..." on peut facilement pousser des mises à jour facilement aux acheteurs

Quelques caractéristiques:

  • Lier votre dépot (à partir de GitHub ou Bitbucket)
  • Options de prix, licence, etc ...
  • Options de paiement avec Stripe
  • Accès privé à votre dépot Git
  • Snipets of any kind

Vos clients seront désormais en mesure de fusionner votre code avec leur propre intégration. De cette façon, ils seront en mesure de rester à jour sans aucune différence!

je vous invite à y jeter un coup d’œil 😉
